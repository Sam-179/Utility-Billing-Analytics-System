Day 2: The Billing Brain (Slabs & Logic)

Slab-Based Math: Created a pricing system where the cost per unit changes based on how much you use (higher usage costs more per unit).

Tier 1: 0 – 100 units at ₹5/unit.

Tier 2: 101 – 300 units at ₹7/unit.

Tier 3: Above 300 units at ₹10/unit.

Fixed Fees: Added a ₹100 service charge to every bill to make it look like a real utility invoice.

Functions: Organized the math into a reusable "function," making the code much easier to manage.
